\### derail

[Merriam-Webster](https://www.merriam-webster.com/dictionary/derail)

- to cause to run off the rails
- to obstruct the progress of : frustrate
    - security problems derailed the tour
- to upset the stability or composure of
    - divorce … can seriously derail an employee
    - — Joanne Gordon
- to leave the rails

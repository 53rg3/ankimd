\### sloppy

[Merriam-Webster](https://www.merriam-webster.com/dictionary/sloppy)

- wet so as to spatter easily : slushy
    - a sloppy racetrack
- wet or smeared with or as if with something slopped over
- slovenly, careless
    - a sloppy dresser
    - did sloppy work
- disagreeably effusive
    - sloppy sentimentalism
